# Gateway AI Systems - Website Clone

This is a pixel-perfect clone of https://gatewayaisystems.com/ built with React.

## Project Structure

```
/app/frontend/src/
├── App.js                          # Main app component
├── App.css                         # Global styles
├── components/
│   ├── Navigation.jsx              # Fixed navbar with mobile menu
│   ├── Navigation.css
│   ├── Hero.jsx                    # Hero section with AI background
│   ├── Hero.css
│   ├── FocusAreas.jsx              # Trucking & Real Estate focus cards
│   ├── FocusAreas.css
│   ├── Mission.jsx                 # Mission section with CTA buttons
│   ├── Mission.css
│   ├── Products.jsx                # 4 product cards
│   ├── Products.css
│   ├── Implementation.jsx          # AI implementation section
│   ├── Implementation.css
│   ├── Hardware.jsx                # Hardware kiosk section
│   ├── Hardware.css
│   ├── Stakeholders.jsx            # Stakeholders identification
│   ├── Stakeholders.css
│   ├── FinalMission.jsx            # Final mission statement
│   ├── FinalMission.css
│   ├── Footer.jsx                  # Footer with contact info
│   └── Footer.css
```

## Design Principles Followed

- No dark/vibrant gradients on buttons (subtle blue gradient only)
- Professional color scheme (blue #1e3a8a to #3b82f6)
- SVG icons from Lucide React (no emoji icons)
- Clean typography with Inter font family
- Generous whitespace and padding
- Smooth hover animations and transitions
- Mobile-responsive with hamburger menu

## Color Palette

- Primary Blue: #1e3a8a
